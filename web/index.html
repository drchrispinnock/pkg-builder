Octez packages

These packages are provided by the Tezos Foundation and are currently under test. They are different from the official packages supplied by Nomadic Labs, which are being deprecated.

For Debian and Ubuntu, you import the GPG key and use the apt repository. curl -s "https://apt.tzinit.org/keys/tzinit.asc" | \ sudo gpg --dearmor -o /etc/apt/keyrings/tzinit.gpg echo "deb [signed-by=/etc/apt/keyrings/tzinit.gpg] https://apt.tzinit.org/debian trixie main" | \ sudo tee /etc/apt/sources.list.d/tzinit-octez.list sudo apt update sudo apt install -y octez-baker Replace debian with ubuntu and trixie with bookworm, etc.
