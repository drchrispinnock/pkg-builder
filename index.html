Unofficial packages

If you are new here, use these packages. If you have previously used the "unoff" packages, please deinstall them and start again with these ones.

  • debian-12 - Debian 12 (amd64/x86_64)
  • debian-12-arm64 - Debian 12 (arm64)
  • ubuntu-2204-lts Ubuntu 22 (amd64/x86_64)
  • ubuntu-2310 Ubuntu 23 (amd64/x86_64)
  • rpi4 - Raspberry Pi 4
  • rpi5 - Raspberry Pi 5
  • rocky-linux-9 - Rocky Linux 9 (amd64/x86_64)
  • aws2023 - AWS Linux 2023 (amd64/x86_64)
