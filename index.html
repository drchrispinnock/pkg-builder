Unofficial packages

If you are new here, use these packages. If you have previously used the "unoff" packages, please deinstall them and start again with these ones.

  • debian-11 - Debian 11 (amd64/x86_64)
  • debian-12 - Debian 12 (amd64/x86_64)
  • debian-12-arm64 - Debian 12 (arm64)
  • ubuntu-2004-lts Ubuntu 20 (amd64/x86_64)
  • ubuntu-2204-lts Ubuntu 22.0 (amd64/x86_64)
  • rpi - Raspberry Pi 4
  • rocky-linux-8 - Rocky Linux 8 (amd64/x86_64)
  • rocky-linux-9 - Rocky Linux 9 (amd64/x86_64)
  • centos-stream-8 - CentOS 8 (amd64/x86_64) - upto 19.1
  • centos-stream-9 - CentOS 9 (amd64/x86_64) - up to 19.1
